# 隐私政策 / Privacy Policy

**最后更新日期 / Last Updated**: 2025-12-09

## 中文版本

### 数据收集

GIF 播放控制器插件不收集、存储或传输任何用户个人信息或浏览数据。

### 数据存储

插件仅在浏览器本地存储中保存以下设置信息:
- 自动加载 GIF 的开关状态

这些数据完全存储在您的本地浏览器中,不会上传到任何服务器。

### 权限说明

插件请求的权限及其用途:

- **activeTab**: 用于在当前标签页中检测和控制 GIF 图片
- **storage**: 用于在本地保存用户的设置偏好
- **host_permissions (<all_urls>)**: 用于在所有网页上检测和处理 GIF 图片

### 第三方服务

本插件不使用任何第三方服务,不包含任何追踪代码或分析工具。

### 数据安全

由于本插件不收集任何数据,因此不存在数据泄露的风险。

## English Version

### Data Collection

The GIF Player Controller extension does not collect, store, or transmit any personal information or browsing data.

### Data Storage

The extension only saves the following settings in browser local storage:
- Auto-load GIF toggle state

This data is stored entirely in your local browser and is never uploaded to any server.

### Permission Explanation

Permissions requested by the extension and their purposes:

- **activeTab**: Used to detect and control GIF images in the current tab
- **storage**: Used to save user preference settings locally
- **host_permissions (<all_urls>)**: Used to detect and process GIF images on all web pages

### Third-Party Services

This extension does not use any third-party services and contains no tracking code or analytics tools.

### Data Security

Since this extension does not collect any data, there is no risk of data breaches.
