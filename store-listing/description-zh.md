# GIF 播放控制器

## 简短描述
智能控制网页 GIF 动图播放,支持暂停、进度拖动和自动加载管理

## 详细描述

### 功能特点

🎮 **完整的播放控制**
- 为网页上的 GIF 图片添加播放/暂停按钮
- 精确的进度条拖动,随时跳转到任意帧
- 流畅的播放体验,无卡顿

⚡ **智能加载管理**
- 自动检测页面 GIF 数量
- 当页面包含超过 4 个 GIF 时,显示播放按钮而非自动加载
- 可在设置中自定义是否自动加载 GIF
- 节省流量和系统资源

🎯 **智能过滤**
- 自动忽略小于 200x100 像素的小图标
- 只处理真正需要控制的 GIF 动图

🎨 **优雅的界面**
- 简洁美观的控制条设计
- 鼠标悬停时显示,不影响浏览体验
- 支持深色和浅色主题

### 使用场景

- 浏览包含大量 GIF 的社交媒体网站
- 查看技术文档中的动画演示
- 控制论坛和博客中的动图播放
- 节省移动设备流量

### 隐私保护

本插件完全在本地运行,不收集任何用户数据,不上传任何信息到服务器。所有设置都保存在浏览器本地存储中。
